# 机票预定系统  
课设..一个简单的CURD项目.
## 运行环境：
操作系统：WIN10    
IDE: Intelli J  
开发工具包：JDK 1.8.0_172  
数据库：MYSQL 5.7   
GUI: JAVAFX    
## 使用教程  
1.安装上述环境。    
2.用intellij打开项目  
2.导入lib目录下的两个jar包  
3.在Project Structure 中配置JDK版本 选择1.8.0_172  
4.在MysqlUtils中配置数据库账号和密码  修改USERNAME与PASSWORD  要与安装的数据库相同  
5.导入提供数据库脚本airlineticket.sql  
6.运行项目
